HTML Etiketleri ve Görevleri

Temel Yapı

  • <html> : Sayfanın başlangıcını ve sonunu belirtir.
  • <head> : Sayfa hakkında teknik bilgileri içerir.
  • <title> : Tarayıcı sekmesindeki başlığı belirler.
  • <body> : Sayfanın tüm görünen içeriğini kapsar.

Metin ve Başlıklar

  • <h1> - <h6> : Başlıkları tanımlar (En büyük h1).
  • <p> : Paragraf oluşturur.
  • <br> : Bir alt satıra geçer.
  • <hr> : Yatay bir çizgi çeker.
  • <b> : Metni kalın yapar.
  • <i> : Metni eğik (italik) yapar.
  • <u> : Metnin altını çizer.
  • <strong> : Önemli metni kalın belirtir.
  • <em> : Vurgulu metni eğik belirtir.
  • <small> : Metni daha küçük yapar.
  • <mark> : Metni işaretler (arkasını boyar).
  • <del> : Üstü çizili metin (silinmiş).
  • <ins> : Altı çizili metin (eklenmiş).
  • <sub> : Alt simge yazar.
  • <sup> : Üst simge yazar.
  • <pre> : Kodlandığı gibi (boşluklarla) görünen metin.

Listeler

  • <ul> : Maddeli (sırasız) liste başlatır.
  • <ol> : Numaralı (sıralı) liste başlatır.
  • <li> : Liste elemanını tanımlar.
  • <dl> : Tanımlama listesi.
  • <dt> : Tanımlama terimi.
  • <dd> : Terim açıklaması.

Bağlantılar ve Medya

  • <a> : Köprü (link) oluşturur.
  • <img> : Resim ekler.
  • <audio> : Ses dosyası ekler.
  • <video> : Video ekler.
  • <source> : Medya kaynaklarını tanımlar.
  • <iframe> : Başka sayfayı içine gömer.

Tablolar

  • <table> : Tablo oluşturur.
  • <tr> : Tablo satırı.
  • <td> : Tablo hücresi.
  • <th> : Tablo başlık hücresi.
  • <thead> : Tablo üst bölgesi.
  • <tbody> : Tablo gövde bölgesi.
  • <tfoot> : Tablo alt bölgesi.

Form Elemanları

  • <form> : Kullanıcı girişi için alan oluşturur.
  • <input> : Veri giriş alanı.
  • <textarea> : Çok satırlı metin alanı.
  • <button> : Tıklanabilir düğme.
  • <select> : Açılır liste.
  • <option> : Liste seçeneği.
  • <label> : Giriş elemanı için açıklama.

Bölümler (Semantik)

  • <div> : Genel amaçlı bölüm.
  • <span> : Küçük metin parçası alanı.
  • <header> : Başlık kısmı.
  • <footer> : Alt bilgi kısmı.
  • <section> : Sayfa bölümü.
  • <article> : Makale/Yazı alanı.
  • <aside> : Kenar çubuğu.
  • <nav> : Menü navigasyon alanı.
